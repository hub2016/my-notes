不常用或不易记的Css代码

  • 单行文字省略号： overflow:hidden;text-overflow:ellipsis;white-space:nowrap;
  • 多行文字省略号line-clamp（非标准CSS属性，必须与其他属性配合使用）： overflow:hidden;text-overflow:ellipsis;display: -webkit-box;-webkit-line-clamp:2;-webkit-box-orient:vertical;
  • 清除浮动的3种方式： 浮动后的元素加：clear:both; 浮动的父元素加：width:100%;overflow:hidden; 浮动的父元素加：{zoom:1}
    :after{content:'';display:block;height:0;clear:both;visibility:hidden}
  • 带透明度遮罩层写法： border:none;background-color:#000;opacity:.8;zoom:1;filter:alpha(opacity=80);margin:0;padding: 0;width:100%;height:100%;top:0;left:0;cursor:auto;position:fixed;_position:absolute
  • placeholder颜色自定义： ::-webkit-input-placeholder{color:#f00;} /* WebKit browsers */
    :-moz-placeholder{color:#f00;} /* Mozilla Firefox 4 to 18 */
    ::-moz-placeholder{color:#f00;} /* Mozilla Firefox 19+ */
    :-ms-input-placeholder{color:#f00;} /* Internet Explorer 10+ */
  • Chrome字体大小小于12px： -webkit-text-size-adjust:none; /*新版已无效*/ display:block;font-size:12px;-webkit-transform:scale(0.8);
  • 禁止选择文本（试试选择本页面的标题，主要用于移动端）： -webkit-user-select:none;-moz-user-select:none;user-select:none;
  • iOS滚动： -webkit-overflow-scrolling:touch;
  • 自定义被选中文本的背景颜色、文字颜色： ::-moz-selection{background:#FE6E66;color:#FFF;}::selection{background:#FE6E66;color:#FFF;}
  • 英文字大写、小写、首字大写、将小写字母转化为小型大写字母 text-transform:uppercase;text-transform:lowercase;text-transform:capitalize;font-variant:small-caps;
  • 文字阴影特效
