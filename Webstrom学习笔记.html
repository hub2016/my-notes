• 设置默认浏览器：File > Settings > Tools > Web Browsers
  • 设置支持PHP：File > Settings > Editor > File Types > 上面选择PHP Files，下面增加规则：*.php
  • 设置自动换行：File > Settings > Editor > General > 勾选 Use soft wraps in editor
  • 配置支持SVN：下载安装slikSVN：https://sliksvn.com/download/

说明：部分电脑在与F*快捷键同时操作时需额外按Fn按钮
  • 常用快捷键（界面操作）：
  • Alt+←：向左切换代码选项卡（循环）
  • Alt+→：向右切换代码选项卡（循环）
  • Ctrl+F4：关闭当前选项卡
  • Ctrl+Shift+F12：代码区域显示最大化
  • 常用快捷键（代码编辑）：
  • F2：定位错误或警告（向后）
  • F3：查找下一个
  • Tab：增加缩进或生成代码
  • Alt+F1：定位文件
  • Alt+Insert：新建文件、目录等
  • Shift+F2：定位错误或警告（向后）
  • Shift+F3：查找上一个
  • Shift+F6：重命名/重构
  • Shift+Tab：减少缩进
  • Shift+Enter：另起一行，无论光标在什么位置
  • Ctrl+/：当前整行或部分选中的多行添加注释
  • Ctrl+B：打开关标处的类、方法或文件
  • Ctrl+D：复制当前行或选中块
  • Ctrl+E：打开最近的文档
  • Ctrl+F：查找
  • Ctrl+G：查找行
  • Ctrl+P：方法参数提示
  • Ctrl+R：替换
  • Ctrl+W：选中单词
  • Ctrl+X：删除当前行或选中文字
  • Ctrl+Y：删除当前行或选中多行（整行）
  • Ctrl+Z：撤销
  • Ctrl+←：以单词为边界光标向前跳
  • Ctrl+→：以单词为边界光标向后跳
  • Ctrl+Alt+L：格式化代码（可以选中部分格式化或不选全格式化）（与QQ锁定快捷键冲突，可修改QQ快捷键）
  • Ctrl+Alt+Enter：在当前行前面新建一行，无论光标在什么位置
  • Ctrl+Shift+/：仅选中的部分文字添加注释
  • Ctrl+Shift+F7：高亮显示所有该文本，按Esc取消
  • Ctrl+Shift+F10：执行文件
  • Ctrl+Shift+C：拷贝文件路径（当前文件或目录选中文件 具体看哪个获得焦点）
  • Ctrl+Shift+F：指定目录内查找
  • Ctrl+Shift+R：指定目录内替换
  • Ctrl+Shift+V：打开历史复制记录
  • Ctrl+Shift+Z：恢复
  • Ctrl+Shift+↑：代码上移一行
  • Ctrl+Shift+↓：代码下移一行
  • Ctrl+Shift+Enter：在当前行后面新建一行，无论光标在什么位置
  • Ctrl+Alt+Shift+C：拷贝文件名及所在行
