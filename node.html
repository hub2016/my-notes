模块和包
  • 模块：一个实现某些特定功能的文件，以实现模块化编程
    ——通过 require(模块名)引入模块
    ——模块中的功能(如：变量、函数)通过赋给 exports 对象的某个属性提供给调用者使用
  • 包：就是一个文件夹，它将模块封装起来，用于发布、更新、依赖管理和版本控制
    ——通过 package.json 来描述包想信息：入口文件，依赖的外部包等等
    ——通过 npm install 命令来安装包，并通过 require 使用包
    ——通过 npm remove 命令来删除包
